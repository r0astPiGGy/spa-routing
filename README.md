# spa-routing

Список пользователей и их альбомов фотографий. Создан используя SvelteKit и Tailwind.


## Установка

```bash
git clone https://github.com/r0astPiGGy/spa-routing
cd spa-routing
```

```bash
npm install
npm run build
npm run preview
```
